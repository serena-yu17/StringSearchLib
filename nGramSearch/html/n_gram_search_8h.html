float *const  weight = NULL,
const uint16_t  gSize = 3 
)

Index the library based on a string array of key, and another array of additional text, e.g. description.

Parameters
guidA unique id for the indexed library
wordsWords to be searched for. For each row, the first word is used as the master key, in which the row size is rowSize. All rows are flattened into a 1D-array, and can be extracted based on rowSize. In a search, all queries of the words in a row will return the master key.
sizesize of the words
rowSizesize of each text rows of words.
weightA list of weight values for each key. It should be at least as long as the number of rows, i.e. size / rowSize.
gSizesize of grams to be created. Default 3.
